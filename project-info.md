# 🎬 AI直播切片工具 - 项目信息

## 📋 项目概述
这是一个专业的AI驱动直播切片工具，专门支持抖音快手直播抓取、自动语音识别、字幕生成和视频切片。

## 🛠️ 技术栈
- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite
- **样式方案**: Tailwind CSS
- **图标库**: Lucide React
- **开发工具**: ESLint + TypeScript

## 📁 项目结构
```
ai-livestream-slicer/
├── src/
│   ├── App.tsx          # 主应用组件
│   ├── main.tsx         # 应用入口
│   ├── index.css        # 全局样式
│   └── vite-env.d.ts    # Vite类型定义
├── public/              # 静态资源
├── package.json         # 项目配置
├── vite.config.ts       # Vite配置
├── tailwind.config.js   # Tailwind配置
├── tsconfig.json        # TypeScript配置
└── README.md           # 项目说明

```

## 🚀 本地开发命令
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览构建结果
npm run preview

# 代码检查
npm run lint
```
